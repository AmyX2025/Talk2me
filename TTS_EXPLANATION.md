# TTS 和字幕说明

## 🤔 什么是TTS？

**TTS = Text-to-Speech（文本转语音）**

简单来说：**TTS就是让AI"说话"的技术**

### 举个例子：

**原文本：**
```
"What do you think about learning English?"
```

**经过TTS处理后：**
- 🔊 **你听到**：AI用语音说出这句话（就像真人说话一样）
- 📺 **屏幕上显示**：同时显示这句话的文字（字幕）

## 📺 字幕是什么？

**字幕 = 屏幕上显示的文字**

在我们的应用中，字幕有几种用途：

### 1. 跟读时的字幕
```
[播放音频] "Welcome to All Ears English"
[显示字幕] Welcome to All Ears English
```
**作用**：让你看到要读什么，方便跟读

### 2. AI说话时的字幕
```
[AI说话] "What do you think about this?"
[显示字幕] What do you think about this?
```
**作用**：让你看到AI说了什么，方便理解

### 3. 你说话时的字幕（实时识别）
```
[你说话] "I think it's good"
[显示字幕] I think it's good
```
**作用**：让你确认自己说了什么，Whisper识别出的文字

## 🎯 在我们的应用中

### 场景1：跟读练习

```
1. 播放原播客音频 🔊
   ↓
2. 同时显示字幕 📺
   "Welcome to All Ears English"
   ↓
3. 你跟着读（不需要录音）
   ↓
4. 可以暂停、复读、选择片段
```

**不需要TTS**：因为播放的是原播客的音频

### 场景2：AI Role Play对话

```
1. AI生成回复（文字）
   "What do you think about this?"
   ↓
2. TTS把文字转成语音 🔊
   [你听到AI说话]
   ↓
3. 同时显示字幕 📺
   [你看到文字]
   ↓
4. 你说话回答
   ↓
5. Whisper识别你的语音 → 显示文字
   ↓
6. 重复步骤1-5
```

**需要TTS**：因为AI生成的是文字，需要转换成语音让你听到

## 💡 总结

| 功能 | 需要TTS？ | 需要字幕？ | 说明 |
|------|----------|-----------|------|
| 跟读练习 | ❌ 不需要 | ✅ 需要 | 播放原音频，显示字幕 |
| AI对话 | ✅ 需要 | ✅ 需要 | AI文字→TTS转语音+显示字幕 |
| 语音识别 | ❌ 不需要 | ✅ 需要 | 显示识别出的文字 |

## 🔧 技术实现

### TTS（文本转语音）
- **技术**：本地TTS（coqui-tts 或 piper）
- **作用**：把AI生成的文字转换成语音
- **成本**：免费（本地运行）

### 字幕显示
- **技术**：前端React组件
- **作用**：在屏幕上显示文字
- **成本**：免费（前端功能）

## ❓ 常见问题

### Q: TTS和字幕是分开的吗？
A: 不是。TTS生成语音，字幕显示文字，它们是**配合使用**的：
- AI说话时 = TTS生成语音 🔊 + 显示字幕 📺
- 你跟读时 = 播放原音频 🔊 + 显示字幕 📺

### Q: 为什么需要字幕？
A: 因为：
1. 帮助你理解（看到文字更容易理解）
2. 方便跟读（知道要读什么）
3. 确认识别（看到Whisper识别出的文字）

### Q: 字幕是自动生成的吗？
A: 是的：
- 跟读时：Whisper识别原音频生成的文字
- AI对话时：AI生成的文字直接显示
- 你说话时：Whisper实时识别你的语音

### Q: 可以关闭字幕吗？
A: MVP阶段会一直显示字幕（帮助学习）。后续可以添加开关。
